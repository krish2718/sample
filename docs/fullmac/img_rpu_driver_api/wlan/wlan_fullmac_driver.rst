=======================
WLAN FullMAC Driver API
=======================

Imagination's WLAN FullMAC Driver exposes the following APIs:

        1. UMAC API - UMAC API covers the various host API constructs (structures, functions etc) which would be necessary to interact with Imagination's UMAC firmware running in the RPU.
        
        2. OSAL API - OSAL API covers the various host API constructs (structures, functions, Ops etc) which would be necessary to interface with different OSes.
